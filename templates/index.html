❓ How do I get these IDs? ▼

🤖 Bot Token

1. Go to Discord Developer Portal
2. Create New Application → Go to "Bot" section
3. Click "Reset Token" and copy it
4. Important: Enable MESSAGE CONTENT and SERVER MEMBERS intents!

🏠 Server ID

1. Enable Developer Mode: Settings → Advanced → Developer Mode
2. Right-click the server icon → "Copy Server ID"

👤 User ID (Optional Filter)

Leave empty to download ALL messages!

Only use this if you want to filter specific users:
1. Enable Developer Mode: Settings → Advanced
2. Right-click a user → "Copy User ID"
3. For multiple users: 123456,789012,345678

📨 Invite the Bot

Go to OAuth2 → URL Generator in the Developer Portal.
Select: bot scope
Permissions: Read Message History, Read Messages/View Channels
Use the generated URL to invite the bot!
